# COMPARATIVE-ANALYSIS-OF-BFS-DFS-USING-ADJACENCY-LIST-AND-CSR-REPRESENTATIONS
Comparative performance analysis of BFS and DFS using Adjacency List and CSR representations for scalable graph traversal.

This project evaluates the performance of Breadth First Search (BFS) and Depth First Search (DFS) using two graph representations:

1. Adjacency List
2. Compressed Sparse Row (CSR)

## Features
- BFS and DFS implementation in C++
- Graph generation for testing
- Execution time comparison
- Performance visualization using Python

## Results
CSR representation shows better cache locality and faster traversal for large graphs.

- C++
- Python (Matplotlib)
